## Preliminary Technical Roles

Technical Roles:

* **Documentation Editor (i.e. Documentation Demon):** Responsible for final reviews and edits to all documentation.

* **Backend Developer (i.e. Backend Buster):** Responsible for reviewing backend code and designing the backend structure.

* **Frontend Developer (i.e. Frontend Freak):** Responsible for reviewing frontend code and creating the frontend design.

* **Database Developer (i.e. Database Donnie):** Responsible for designing the database structure and developing the database client system.

* **Version Control Manager (i.e. GitHub Grub):** Responsible for managing the GitHub repo, pull requests, versions, branches, and ensuring that each team member maintains their own cloned repo.

* **Presentation Editor (i.e. Presentation Punter):** Responsible for final reviews and edits to all presentations.
